# 监管者项目报告

日期：2026-06-16  
版本：v1.0.0 第一版  
面向读者：看完视频过来的粉丝、对 AI 工具感兴趣的普通 Mac 用户

## 这是什么

监管者是一个 Mac 端 AI 摸鱼监工。你自己设置黑名单，自己开启监督。它发现你打开了黑名单里的网站或 App，就会弹出桌面悬浮小组件，用 AI 语音当场吐槽你。

它的重点不是“效率管理”，而是“被 AI 抓包”的冲突感和节目效果。你可以把它当成一个更好玩的自律提醒器，也可以拿来录屏做挑战视频。

## 第一版能玩什么

- 添加摸鱼网站，例如 B 站、YouTube、X/Twitter、Reddit。
- 添加本机 App，例如 Steam、游戏、视频软件或聊天工具。
- 开启监督后，命中黑名单会触发 AI 吐槽。
- 按住快捷键语音反驳，让 AI 继续回应。
- 说“监督我接下来的 40 分钟”，快速开始一个倒计时监督任务。
- 选择中文或英文监督语言。
- 调整吐槽强度和声音。
- 查看今天的抓包记录。

## 怎么下载

目前只支持 Mac。

- 下载文件：`Hunter.dmg`
- 下载位置：[GitHub Releases](https://github.com/FerryCorleone/Hunter/releases/latest)
- 系统要求：macOS 14 Sonoma 或更高版本

暂不支持 Windows、iPhone、iPad、Android。

## 怎么安装

1. 下载 `Hunter.dmg`。
2. 双击打开。
3. 把 `监管者.app` 拖到 `Applications`。
4. 第一次打开时，如果 macOS 拦截，右键 App 选择“打开”。
5. 按提示允许麦克风、浏览器自动化和通知。

## 模型配置说明

监管者需要三种 AI 能力：

- ASR：听懂你说话。
- LLM：生成吐槽和回击。
- TTS：把吐槽念出来。

你不需要懂技术细节。打开设置里的“AI”，每一块按这个顺序来：

1. 选厂商。
2. 选模型，默认推荐一般就能用。
3. 填 API Key。
4. 点输入框后面的保存/更新。
5. 点测试按钮确认能用。

如果你没有 API Key，可以先去对应模型厂商官网申请。监管者不会自带模型账号，也不会把任何密钥提交到仓库。

## 隐私和边界

- 黑名单、历史记录和设置默认保存在本机。
- API Key 存在本机，不写进 GitHub 仓库。
- 浏览器 URL 只在监督开启且前台是支持的浏览器时读取。
- 只有命中黑名单或语音对话时，才把必要文本发给你选择的模型服务。
- 这是个人自愿使用工具，不做隐身监控、远程上报或公司管控。

## 当前限制

- 只支持 macOS。
- 第一版只提供 DMG 安装包。
- 云端 ASR / LLM / TTS 需要用户自己的 API Key。
- 浏览器 URL 读取依赖 macOS 自动化权限。
- 第一次运行可能需要手动允许麦克风和浏览器自动化权限。

## 给看视频来的朋友

最快试玩路线：

1. 安装 App。
2. 在黑名单里加一个你最常刷的网站。
3. 在 AI 里填好模型 Key。
4. 开始 15 分钟监督。
5. 打开黑名单网站，录屏看它怎么抓你。

如果你只是想看热闹，也可以把它当成一个“桌面 AI 监工玩具”。如果你真的想自律，它会稍微烦你一点，这正是它存在的意义。
